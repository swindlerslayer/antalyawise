# Antalya Wise 🌊

Antalya'da havalimanı transferi, günlük turlar, otel rezervasyonları ve seyahat rehberi sunan premium, modern ve hızlı seyahat platformu.

## 🚀 Hızlı Başlangıç

Projeyi yerel bilgisayarınızda çalıştırmak ve geliştirmek için aşağıdaki adımları takip edebilirsiniz.

### Gereksinimler

- [Node.js](https://nodejs.org/) (v18 veya üzeri önerilir)

### Kurulum

1. Gerekli bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu (Vite) başlatın:
   ```bash
   npm run dev
   ```

3. Tarayıcınızda açılan adresten (genellikle `http://localhost:5173`) projeyi canlı olarak görüntüleyin ve düzenlemeler yapın.

## 🛠️ Proje Yapısı

- `index.html` - Canlı, etkileşimli, responsive ve şık tasarıma sahip ana sayfa.
- `package.json` - Vite geliştirme sunucusu ve proje bağımlılıkları.
- `README.md` - Proje başlangıç ve kullanım kılavuzu.
